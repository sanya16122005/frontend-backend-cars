О приложении

🚗 Cars Tasks — PWA для управления задачами по автомобилям.

  • Версия: 1.5.0
  • Архитектура: App Shell — каркас грузится мгновенно, контент динамически
  • Хранение: localStorage
  • Кэширование статики: Cache First через Service Worker
  • Кэширование контента: Network First с фолбеком на главную
  • Работает офлайн, устанавливается на главный экран

Откройте DevTools → Application → Cache Storage и убедитесь, что присутствуют два кэша: car-shell-v1 и car-dynamic-v1.
